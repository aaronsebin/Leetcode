# LeetCode Solutions

A personal collection of LeetCode problem solutions, written as I work through
problems across different topics and difficulty levels.
Each solution includes the problem statement and the final code.

## Structure
Each problem lives in its own folder named after the problem, containing the
solution file and any notes.

## Goals
- Build consistency in problem solving
- Understand the "why" behind each approach, not just the answer
- Progress from brute force to optimised solutions step by step

## Topics Covered
Arrays, Hash Maps, Two Pointers, Sliding Window, Linked Lists, Trees, 
Dynamic Programming, and more as I go.

## Languages
Python
